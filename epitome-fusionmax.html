Description / Indications

Epitome Fusionmax is a flowable bulk-fill composite for use as base for class I and II direct restorations on posterior teeth. It allows large increments (up to 4mm thick). It adapts easily to irregular cavities.

Applicability

  • Restorations of class III, IV, V cavities
  • Root surface caries restorations
  • For initial placement in class I and II cavities
